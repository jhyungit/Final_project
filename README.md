# 🍽 콘텐츠 기반 추천 시스템

## 📌 프로젝트 소개
사용자의 선호 정보를 기반으로 유사한 콘텐츠를 추천하는  
**콘텐츠 기반 추천 시스템(Content-Based Recommendation System)** 입니다.  
(T-academy 기업 연계 프로젝트 / 실제 기업 데이터 활용)

---

## 🎯 개발 목적
- 사용자 맞춤형 추천 기능 구현 경험
- 데이터 기반 서비스 로직 설계 및 구현
- 머신러닝 기반 추천 시스템 이해

---

## 🛠 기술 스택
- Python
- Pandas
- Scikit-learn

---

## 🔍 주요 기능
- 사용자 특성 기반 콘텐츠 추천
- 유사도 계산을 통한 추천 알고리즘 구현
- 데이터 전처리 및 피처 엔지니어링

---

## ⚙️ 구현 방법

### 1. 데이터 전처리
- 결측치 처리 및 데이터 정제
- 추천에 필요한 feature 추출

### 2. 추천 모델 구성
- Content-Based 기반 추천
- CB + CF, ML, DL 모델을 결합하여 성능 보완

### 3. 유사도 기반 추천
- 코사인 유사도(Cosine Similarity)를 활용하여 추천 수행

---

## 📊 결과 및 성과

- 기존 수작업 추천 프로세스 자동화
- T-academy Big-data 과정 **우수팀 선정**
- 약 2만 명 규모 기업(SPARKPLUS)과 계약 성사
